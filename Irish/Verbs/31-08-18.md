# Verbs

## Ól - To Drink

| **Aimsir Chaite** | **Aimsir Láithreach** | **Aimsir Fháistineach** |
| ----------------- | --------------------- | ----------------------- |
| D'ól mé           | Ólaim                 | Ólfaidh mé              |
| D'ólamar          | Ólaimid               | Ólfaimid                |
| Níor ól           | Ní ólaim              | Ní ólfaidh              |
| Ar ól             | An ólann              | An ólfaimid             |
